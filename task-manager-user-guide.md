# Task Manager User Guide

## Overview

Task Manager is a web application designed to help users manage tasks and organize their workflow.

---

## Getting Started

1. Open the application  
2. Sign in to your account  
3. Go to the dashboard  

---

## Creating a Task

1. Click **"New Task"**  
2. Enter the task title  
3. Add a description (optional)  
4. Click **"Save"**

---

## Editing a Task

1. Select a task  
2. Click **"Edit"**  
3. Update the information  
4. Click **"Save"**

---

## Deleting a Task

1. Select a task  
2. Click **"Delete"**  
3. Confirm the action  

---

## Task Status

Tasks can have the following statuses:

- New  
- In Progress  
- Completed  

To change status:

1. Open the task  
2. Select status  
3. Save  

---

## Search and Filtering

You can:

- search by task name  
- filter by status  

---

## Common Issues

### Task is not saved

- Check required fields  
- Check internet connection  

---

### Changes not visible

- Refresh page  
- Log in again  

---

## Tips

- Use clear task titles  
- Update status regularly  
- Remove completed tasks  

---

## Conclusion

Task Manager helps users stay organized and manage tasks effectively.
